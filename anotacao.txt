criar projeto (abra a pasta com o nome nesse estilo nome-nome no vs e copie esse cod)

	npx create-react-app name-name


startar a sua aplicação no terminal

	npm start

instale a extensao do vs code ES7 pra poder rdar comandos de auto generete elements

rafce   criar uma funcao pra gerar um elemento


Install
Inside VSCode, press Ctrl+P, and enter:

ext install diegolincoln.vscode-styled-components

======================================================


Instalar a dependencia dos styled components

	npm install --save styled-components
	npm i styled-components

Instalar a dependencia dos react router

	npm install react-router-dom

Instalar a dependencia do react icons

	npm install react-icons --save

Instalar a dependencia do react scroll

	npm install react-scroll
